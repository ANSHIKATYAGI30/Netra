# 👁 Netra

Netra is a modern desktop QR Code Scanner built using Python, OpenCV, and CustomTkinter.

## Features

- 📷 Real-time QR code scanning
- 🟩 QR bounding box detection
- 🔗 Automatic URL recognition
- 📋 One-click copy to clipboard
- 🌐 Open scanned URLs in browser
- 💾 Scan history logging
- 🖥 Modern CustomTkinter interface
- ⚡ Threaded camera capture
- 🚫 Duplicate scan prevention

---

## Technologies Used

- Python 3.11+
- OpenCV
- CustomTkinter
- Pillow
- NumPy
- Pandas
- Pyperclip

---

## Installation

Clone the repository:

```bash
git clone https://github.com/<your-username>/Netra.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
